# Modelo 2 — Automação Essencial

Escopo do modelo atual do produto Pés do Paraná.

## Objetivo

Entregar uma operação web funcional para divulgar eventos, captar reservas e organizar a operação básica.

## Entregas do modelo

- agenda pública dinâmica;
- página pública individual do evento;
- reserva online estruturada;
- formulários automáticos;
- lista organizada de participantes;
- painel administrativo simples.

## Fora de escopo

- pagamento online;
- área do participante;
- Clube PP;
- certificados;
- recuperação automática de reserva abandonada;
- avaliação pós-evento;
- histórico do trilheiro;
- automações avançadas de WhatsApp/e-mail.

## Fases

1. Agenda pública dinâmica.
2. Página pública individual do evento.
3. Reserva operacional.
4. Lista de participantes.
5. Comunicação.
